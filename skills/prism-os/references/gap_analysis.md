# Gap Analysis - 素材缺口分析

## 目标

计算命题所需的"证据链"与现有素材的匹配度，输出结构化的 Gap Report。

## 流程

```
命题 → 提取证据链 → 匹配素材 → 计算缺口 → 输出报告
```

## 证据链类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 数据类型 | 统计数据、调研报告 | "AI 对不同岗位的冲击数据" |
| 案例类型 | 真实案例、对比实验 | "教育系统的结构性缺陷案例" |
| 理论依据 | 学术理论、专家观点 | "认知心理学中的元认知理论" |

## Prompt

```
你是内容策划分析师。分析选题需要的证据链，评估现有素材就绪度。

命题：{{THESIS}}
现有素材：{{MATERIALS}}

提取证据链：
1. 数据类型：统计数据、调研报告
2. 案例类型：真实案例、对比实验
3. 理论依据：学术理论、专家观点

返回 JSON：
{
  "evidence_chain": ["证据1", "证据2", "证据3"],
  "matched_materials": [{"evidence": "证据1", "source": "...", "match_score": 0.85}],
  "missing_evidence": ["证据2", "证据3"],
  "gap_score": 0.67,
  "readiness": 0.33,
  "recommendation": "建议补充 2 个关键素材"
}
```

## 计算公式

```
Gap Score = 未匹配证据数 / 总证据数
Readiness = 1 - Gap Score
```

## 输出格式

```json
{
  "evidence_chain": [
    "AI 对不同岗位的冲击数据",
    "提问能力的量化指标",
    "教育系统的结构性缺陷案例"
  ],
  "matched_materials": [
    {"evidence": "AI 对不同岗位的冲击数据", "source": "material_123.md", "match_score": 0.85}
  ],
  "missing_evidence": [
    "提问能力的量化指标",
    "教育系统的结构性缺陷案例"
  ],
  "gap_score": 0.67,
  "readiness": 0.33,
  "recommendation": "建议补充 2 个关键素材后再执行"
}
```

## 阈值建议

| Readiness | 状态 | 建议 |
|-----------|------|------|
| > 0.8 | 素材充足 | 可以开始创作 |
| 0.5-0.8 | 素材不足 | 补充关键素材 |
| < 0.5 | 素材匮乏 | 建议暂缓，先收集素材 |
